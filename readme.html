Running the program

To run the program just run the Main class from the prog package.

Adding a command

The way I wrote the program makes it very simple to add a command.
To add a new command just add a new class that implements the command interface,
then add the following line into the CommandContainer constructor:
hm.put([command], new [NewCommand]());

English Checker Algorithm

The EnglishChecker class implements the first Algorithm suggested in the exercise.
First, we build the English Frequency Table and the given text Frequency table.
then, we compare the two tables and if the variance is below a certain threshold,
then the given text is very likely to be in english.
